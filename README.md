
基于boost::context实现一个简单易用的协程库。

该库有以下特性：

1. 栈自动回溯。未执行完的协程释放时，会自动回溯栈，令C++对象能正确析构。
2. 协程生命周期自动管理。无需显式调用destroy，协程未在执行并没有任何地
   方持有时，会自动释放协程。

常用组件：

1. 简单的调度器，轮循方式
2. 简单的分发器，适用于协程内需要等待接收一个数据的情况

依赖库：

1. boost::context (-lboost_context)
2. boost::intrusive_ptr
